# Customer Retention & Churn Analysis

## Project Objective
Analyze customer churn behavior to identify retention drivers, churn patterns, and customer lifetime trends. The goal is to understand why customers leave and provide recommendations to improve customer retention.

## Dataset
Bank Customer Churn Dataset

## Tool Used
- Microsoft Power BI
- Power Query
- DAX

## Project Files
- BankchurnPBI5version2018-Copy.pbix
- Bank_Customer_Churn_Analysis_Report.pdf

## Key Analysis Performed
- Customer churn rate analysis
- Customer retention analysis
- Age group analysis
- Geography-based churn analysis
- Customer activity analysis
- Credit score analysis
- Product usage analysis

## Key Findings
- Overall churn rate was approximately 20%.
- Inactive customers showed higher churn rates than active customers.
- Customers aged 36–50 contributed significantly to churn.
- Germany recorded higher customer churn compared to other regions.
- Customers with multiple products showed different churn patterns.

## Business Recommendations
- Improve engagement programs for inactive customers.
- Create targeted retention campaigns for high-risk customer groups.
- Monitor customers with high churn probability.
- Develop loyalty and reward programs to improve retention.
